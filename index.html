CookoBBQ

Le livre de recettes français qui manque à votre Ninja Woodfire.

Application mobile gratuite de recettes pour les barbecues électriques Ninja Woodfire, avec mode pas-à-pas pédagogique pensé pour cuisiner en plein air.

Beta Android disponible.

Téléchargement direct : cookobbq-beta.apk (URL stable, toujours la dernière version).

Toutes les releases : github.com/christophebredy/CookoBBQ-Releases/releases.

Installation Android

  1. Télécharge l'APK ci-dessus sur ton téléphone Android.
  2. Autorise l'installation depuis "sources inconnues" si Android te le demande (c'est normal — l'app n'est pas encore sur le Play Store).
  3. Ouvre l'APK pour installer, puis lance CookoBBQ.
