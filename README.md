## Note d'historique

Ce repo a hébergé temporairement les pages auth (`/auth/`) et docs
légaux (`/legal/`) via GitHub Pages. Depuis V0.7.0, ces pages sont
sur **cookobbq.com** (Vercel) avec un style aligné sur la landing.
Les anciennes URLs `christophebredy.github.io/CookoBBQ-Releases/auth/*`
redirigent automatiquement (en préservant le hash URL pour les tokens
Supabase). On retirera les redirects quand on sera sûr qu'aucun
ancien email Supabase ne traîne (~V0.9).
